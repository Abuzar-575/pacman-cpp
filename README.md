# Pac-Man - C++ SFML

A fully functional Pac-Man game built in C++ using SFML library, 
developed as part of Object-Oriented Programming coursework.

## Features
- Classic Pac-Man gameplay with 4 ghosts
- Each ghost has a unique AI strategy:
  - Red (Blinky) - Direct pursuit
  - Pink (Pinky) - Ambush ahead of Pac-Man
  - Blue (Inky) - Patrol strategy
  - Orange (Clyde) - Random movement
- Power pellets that make ghosts frightened
- Lives system and score tracking
- Tunnel wrap-around
- HUD displaying score and lives
- Animated Pac-Man sprites

## Tech Stack
- C++
- SFML (Simple and Fast Multimedia Library)

## OOP Concepts Used
- Inheritance (Character base class)
- Polymorphism (virtual functions)
- Abstract classes (pure virtual methods)
- Strategy design pattern (Ghost AI)
- Encapsulation

## How to Run
1. Install SFML
2. Compile with a C++ compiler linking SFML
3. Run the executable
4. Use arrow keys to move Pac-Man
5. Press R to restart, Escape to quit
